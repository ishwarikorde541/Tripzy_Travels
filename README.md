# Tripzy_Travels
Tripzy Travel is a responsive travel website built using **HTML** and **CSS**. The project showcases a modern travel-themed landing page with an attractive user interface, destination highlights, and a clean design aimed at enhancing the user experience.

## 🚀 Features

- Responsive and user-friendly design
- Modern travel-themed layout
- Attractive navigation bar
- Destination showcase sections
- Clean and organized code structure
- Built entirely with HTML and CSS

## 🛠️ Technologies Used

- HTML5
- CSS3
